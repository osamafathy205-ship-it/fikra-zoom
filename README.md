# Fikra Academy — Virtual Classroom Server

This is the unified backend and frontend server for Fikra Academy, deployed on HuggingFace Spaces.

## 🚀 Deployed URL Format
Once deployed, the student access URL is:
`https://<your-username>-<your-space-name>.hf.space/#/waiting/<meeting-id>`

Example:
`https://osamafathy-fikra-academy.hf.space/#/waiting/BD582373`
